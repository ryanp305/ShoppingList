Shopping List

Heading to the grocery store? Make a shopping list.

How it works:

Type the name of an item in the text box below and click the add to list button to add it to your list. Once an item has been added you can use the '✓' button to cross an item off your list or the 'x' button next to the item. Click the orange reset button below if you want to clear your entire list and start a new one.
